#LuiGi OS
LuiGi OS is a project designed to bring the classic Wii menu experience to the web. It is a full functional remake of the Wii system interface, built to serve as a hub for playing games and running media through a browser.

#What is LuiGi OS?
The project is a detailed simulation of the Wii operating system. It features the signature grid layout, transition effects, and sound design of the original console. It isn't just a static clone; it's a dynamic dashboard that allows you to launch and manage interactive content in a "Virtual Sandbox" environment.

The goal was to create a space that feels like a real console where everything—from the settings to the message board—works within a unified UI.

#Features
Wii-Style Interface: A faithful recreation of the system menu, including the iconic channel grid and bottom bar.

Game Sandbox: Runs applications in isolated frames to keep the OS stable and fast.

Stealth Integration: Designed with "Study.com" masking and educational themes to blend into restricted environments.

Dynamic Content Loading: Uses a custom injection engine to pull game files from different sources while bypassing connection blocks.

System Tools: Includes a working settings menu, message board simulations, and audio engine for the full "Year of Luigi" vibe.

Tech Stack
Frontend: Built using Vanilla JavaScript, HTML5, and CSS3.

OS Logic: A custom-built Layout Engine and Bootloader that handles window management and system transitions.

Bypass Logic: Employs a fetch-and-inject method to render external HTML content directly into the sandbox, avoiding standard iframe refusal errors.
